# demo-sites/

Staging workspace for AI-generated demo sites before publishing to `/demo/[slug]`.

## Structure

```
demo-sites/
  _imports/   — unzip AI-generated ZIPs here for inspection
  _archive/   — move processed ZIPs here when done
```

Files here are NOT deployed to azisunt.net.
They are local working copies only — for inspection, content extraction, and reference.
